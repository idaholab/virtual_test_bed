# VTB Basics Tutorial

In this short tutorial, we will feature how the VTB should be used for the Molten Salt Reactor
multiphysics core simulation using Griffin and Pronghorn.

!alert note
See [tutorials/neams-workbench.md] instead if you would like to use the NEAMS workbench with the VTB on INL HPC.

If we are interested in the Molten Salt Fast Reactor inputs, navigate to the `msr` folder, then the `msfr` folder:

```
cd ~/projects/virtual_test_bed/msr/msfr
```
